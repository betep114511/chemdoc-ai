# ChemDoc AI 作品集项目

ChemDoc AI 是一个面向化学实验室新人、研究生和研发助理的 RAG 知识库问答助手。它允许用户上传论文、实验 SOP、SDS 和实验记录，并基于可追溯引用回答实验条件、试剂风险、异常处理和安全注意事项。

## 项目定位

这个项目适合写进 AI 产品经理简历，因为它同时覆盖三类能力：

- 垂直场景理解：化学文献、SOP、SDS、安全风险。
- AI 产品能力：RAG、Prompt、引用式回答、拒答、人机协同。
- 产品闭环能力：用户痛点、MVP、指标体系、评测集、失败归因和迭代。

## 文件说明

- `prd.md`：完整 PRD，可作为作品集主体材料。
- `user_research.md`：用户画像、访谈问题、需求洞察。
- `competitive_analysis.md`：竞品和替代方案分析。
- `metrics_and_eval.md`：指标体系、评测方法、上线门禁。
- `case_study.md`：面试/作品集可用的一页项目复盘。
- `interview_script.md`：2 分钟、5 分钟、10 分钟项目讲述稿和追问回答。
- `presentation_guide.md`：简历、作品集和现场演示的展示指南。
- `data/eval_set.csv`：可展示的样例评测集。
- `prototype/index.html`：可直接打开的交互原型。

## 推荐作品集展示顺序

1. 先展示 `case_study.md`，让面试官快速理解项目价值。
2. 再展示 `prototype/index.html`，证明不是停留在概念。
3. 如果面试官追问 AI 质量，打开 `metrics_and_eval.md` 和 `data/eval_set.csv`。
4. 如果面试官追问产品过程，打开 `prd.md` 和 `user_research.md`。
